# Startup Marketplace Platform

SolarNest is a modern **React-based** startup marketplace platform where users can explore, buy, sell, and collaborate with various startups and businesses.

This platform features:
- 📈 Marketplace for startups  
- 🏷️ Filters and sorting functionality  
- 📊 Animated statistics section  
- 🧩 Collaboration and partnership cards  
- 🌐 Multi-language-ready (if needed in future)  
- 🎨 Fully responsive design  
- ⚡ Fast, lightweight & scalable

---

## 📂 Features
- Explore startups listed for sale or funding  
- Detailed marketplace cards for each startup  
- Collaboration cards with startup details  
- Animated counters for platform stats  
- Mobile-friendly filters dropdown for easy sorting  
- Beautiful UI with subtle animations  
- Easy-to-customize components

---

## 🛠️ Tech Stack
- **React JS** (Frontend Framework)
- **CSS Modules** (Scoped Styling)
- **React Router DOM** (Routing)
- **GitHub Pages** (Deployment)
- **gh-pages** (Deployment Automation)
